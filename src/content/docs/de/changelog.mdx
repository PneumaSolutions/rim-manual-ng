Versionen von RIM werden durch Überschriften auf Ebene 3 gekennzeichnet. Wesentliche Updates mit mehreren Funktionen für beide Plattformen werden in Unterüberschriften für Mac und Windows auf Ebene 4 unterteilt.
### 3.4.10

* Dieses Update enthält interne Änderungen zur Verbesserung der Zuverlässigkeit.

### 3.4.9

* Windows: Es wurde ein Problem behoben, das bei einigen Computern beim Aktualisieren auf die vorherige Version auftrat.

### 3.4.8

* Implementiert die Erfassung des Remote-Zugriffsmoduls während einer Sitzungsaufzeichnung über das System-Audio des steuernden Computers.
* Implementiert einen Dialog zur Einholung der Zustimmung auf der Zielseite vor Beginn einer Sitzungsaufzeichnung während aller keywordbasierten Sitzungen. Sollte der Zielbenutzer die Aufzeichnungsanfrage ablehnen, wird die Sitzung beendet.
* Implementiert einen Bestätigungsdialog vor dem manuellen Beenden einer Aufzeichnung. Dieser Dialog kann auf Wunsch so konfiguriert werden, dass er nicht angezeigt wird.
* Es ist nicht mehr möglich, eine Aufzeichnung zu starten, während eine Sitzung umgeschaltet wird.
* Implementiert die Möglichkeit, das Standardverzeichnis für Aufzeichnungen sowie mehrere andere Konfigurationsparameter über eine TOML-Datei festzulegen. Anweisungen zum Konfigurieren dieser Optionen finden Sie im Abschnitt über die Sitzungsaufzeichnung.

### 3.4.7
Diese Version führt eine offene Beta-Version der Sitzungsaufzeichnungsfunktion für beide Plattformen ein. Diese Funktion kann von jedem kostenlos genutzt werden, und der Cloud-Dienst wird niemals für die Verarbeitung der Aufzeichnung verwendet. Die Aufzeichnungen werden derzeit im lokalen Videordner des steuernden Benutzers gespeichert, d. h. unter Windows im Ordner „Videos” und unter MacOS im Ordner „Movies”.  
Bitte zögern Sie nicht, uns Feedback zu senden (https://getrim.app/feedback), wenn Sie bei der Verwendung dieser Funktion auf Probleme stoßen oder wenn Sie Kommentare oder Vorschläge zur Verbesserung haben.

Wenn Sie den unbeaufsichtigten Zugriff auf einen oder mehrere Zielcomputer eingerichtet haben, können Sie jetzt eine Verknüpfung erstellen, um direkt auf einen dieser Computer zuzugreifen. Klicken Sie einfach auf die Schaltfläche „Verknüpfung erstellen” unter der Liste der unbeaufsichtigten Ziele, und RIM erstellt eine Verknüpfung auf Ihrem Desktop.
### 0.10.9
RIM funktioniert jetzt auch auf Zielcomputern ohne Audiogerät. In Kombination mit dem bestehenden Fernzugriffsmodul eröffnet dies den Zugriff auf Server und virtuelle Maschinen, auf die zuvor mit einem Screenreader nur über RDP und dessen sehr suboptimale Audioausgabe zugegriffen werden konnte.
### 0.10.8
* Wenn Sie das erste Jahr Ihres RIM Pro-Abonnements noch nicht vorbestellt haben, können Sie dies jetzt ganz einfach über die neue Schaltfläche auf dem Bildschirm „Hilfe anbieten“ tun. Wir führen auch neue Preisoptionen für das Hinzufügen zusätzlicher Controller und gleichzeitiger Kanäle zu einem RIM Pro-Abonnement ein. Sie können bis zu zwei zusätzliche Controller-Benutzer und zwei zusätzliche Kanäle für jeweils 500 US-Dollar pro Jahr hinzufügen. RIM ist während der öffentlichen Beta-Phase weiterhin kostenlos, aber wenn Sie Ihr Abonnement jetzt vorbestellen, können Sie den Dienst nach Ablauf der öffentlichen Beta-Phase am 1. September ohne Unterbrechung weiter nutzen.
* Wir führen Control+Shift+Backspace als alternativen Hotkey zum Öffnen des RIM-Menüs auf beiden Seiten der Verbindung ein. Wir unterstützen weiterhin Windows+Shift+Escape, aber wir planen, Control+Shift+Backspace als einzigen Hotkey für alle Phasen von RIM zu standardisieren.
* Wenn Sie auf dem Bildschirm „Hilfe erhalten“ auf die Schaltfläche „Stattdessen Hilfe bereitstellen“ klicken, legt RIM diese Option nicht mehr als Standard fest, bis Sie sich bei Ihrem RIM-Konto angemeldet haben.
* Einige JavaScript-Fehler, die in seltenen Fällen auftraten, wurden behoben.

### 0.10.7
Sie können nun das Mausrad in einer Remote-Sitzung verwenden.
### 0.10.6
Hoffentlich wurden alle verbleibenden Probleme behoben, die zum Absturz des Installationsprozesses führten.
### 0.10.5
Diese Version enthält einige Änderungen unter der Haube. Insbesondere haben wir Aktualisierungen der Installation und der zugrunde liegenden Pakete vorgenommen, um Probleme zu beheben, die zum Absturz der Installation führten.
### 0.10.4
* Im RIM-Menü gibt es jetzt eine Option, um Control+Alt+Delete an den Zielcomputer zu senden.
* Sie können das RIM-Menü jetzt durch einen Rechtsklick auf die Titelleiste des Sitzungsfensters öffnen. Wir planen, die Option hinzuzufügen, das RIM-Menü auch durch Klicken auf das RIM-Symbol am linken Ende der Titelleiste zu öffnen.

### 0.10.3
* Der Controller kann jetzt eine unbeaufsichtigte Sitzung umschalten.
* Der Controller kann jetzt einen unbeaufsichtigten Zielcomputer umbenennen.
* Es wurde ein Fehler behoben, der manchmal dazu führte, dass die Sitzung nach dem Kopieren in die Zwischenablage nicht mehr reagierte.
* Es wurde ein Fehler behoben, der manchmal dazu führte, dass das Starten des Remote-Zugriffsmoduls mit einem JavaScript-Fehler fehlschlug.

### 0.10.2

### 0.10.1
* Sie können jetzt das erste Jahr Ihres Abonnements für RIM Pro für 999 $ vorbestellen. Der Link zur Vorbestellung wird im Feedback-Fenster angezeigt, wenn Sie eine Remote-Sitzung beenden. Oder Sie können dem direkten Link hier folgen: [RIM Pro vorbestellen.](https://getrim.app/signup/rim_preorder)
* Wenn Sie unbeaufsichtigte Zielcomputer in Ihrem RIM-Konto haben, wird die Schaltfläche zum Starten einer unbeaufsichtigten Sitzung nun zuverlässiger auf dem Bildschirm „Hilfe bereitstellen“ angezeigt.
* Beim Umschalten einer Sitzung kann der ursprüngliche Controller die Sitzung nun zurückumschalten, indem er das RIM-Menü mit Windows+Umschalt+Escape öffnet.
