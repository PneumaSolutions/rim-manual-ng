<Windows>
Im Folgenden finden Sie eine vollständige Tastaturbefehlsreferenz für RIM. Es werden alle Befehle aufgelistet, die globale Tastenkombinationen haben, zusammen mit den entsprechenden Windows-freundlichen Tastenkombinationen.

| Aktion | Befehl |
| ---------------------------------------- | ------------------- |
| Zugriff auf das RIM-Menü | Windows+Umschalt+Rücktaste |
| Sitzung minimieren | Windows+Umschalt+M |
| Sitzung umkehren | Windows+Umschalt+F |
| Sitzung trennen (Controller oder Ziel) | Windows+Umschalt+D |
</Windows>
<MacOS>
Im Folgenden finden Sie eine vollständige Tastaturbefehlsreferenz für RIM. Es werden alle Befehle aufgelistet, die globale Tastenkombinationen haben, zusammen mit den entsprechenden macOS-freundlichen Tastenkombinationen.

    Aktion | Befehl
    --- | ---
    Zugriff auf das RIM-Menü | Wahl+Umschalt+Rücktaste
    Sitzung minimieren | Wahl+Umschalt+M
    Sitzung umkehren | Wahl+Umschalt+F
    Sitzung trennen (Controller oder Ziel) | Wahl+Umschalt+D
</MacOS>
