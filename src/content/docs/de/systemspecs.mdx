Um Remote Incident Manager ausführen zu können, muss Ihr System die folgenden Anforderungen erfüllen oder übertreffen:

<Windows>
  * Windows 10 (X64) oder höher
  * Windows Server 2016 (X64) oder höher
  * Mindestens 4 GB RAM
  * Admin-Rechte für die Erstinstallation (nicht erforderlich für automatische Updates)
</Windows>

<MacOS>
  * Intel oder Apple Silizium
  * macOS Ventura oder höher
  * Mindestens 4 GB RAM
  * Admin-Rechte für die Erstinstallation (nicht erforderlich für automatische Updates)
</MacOS>
