### 3.4.8

* Implémente la capture du module d&#x27;accessibilité à distance pendant l&#x27;enregistrement d&#x27;une session via le système audio de l&#x27;ordinateur de contrôle.
* Implémente une boîte de dialogue de consentement côté cible avant le début d&#x27;un enregistrement de session pendant toute session basée sur des mots-clés. Si l&#x27;utilisateur cible refuse la demande d&#x27;enregistrement, la session prend fin.
* Implémente une boîte de dialogue de confirmation avant l&#x27;arrêt manuel d&#x27;un enregistrement. Cette boîte de dialogue peut être configurée pour ne pas s&#x27;afficher si vous le souhaitez.
* Il n&#x27;est plus possible de démarrer un enregistrement pendant qu&#x27;une session est basculée.
* Implémente la possibilité de définir le répertoire d&#x27;enregistrement par défaut, ainsi que plusieurs autres paramètres de configuration, via un fichier toml. Reportez-vous à la section sur l&#x27;enregistrement de session pour obtenir des instructions sur la configuration de ces options.

N&#x27;hésitez pas à [nous envoyer vos commentaires](https://getrim.app/feedback) si vous rencontrez des problèmes lors de l&#x27;utilisation de cette fonctionnalité ou si vous avez des remarques ou des suggestions pour l&#x27;améliorer.

### 3.3.27

* Cette mise à jour apporte quelques modifications à la façon dont RIM affiche le pointeur de la souris et gère les entrées de la souris.
