Pour que Remote Incident Manager puisse fonctionner, votre système doit remplir ou dépasser les conditions suivantes :

<Windows>
  * Windows 10 (X64) ou version ultérieure
  * Windows Server 2016 (X64) ou version ultérieure
  * Au moins 4 Go de RAM
  * Privilèges d'administrateur pour une fonctionnalité complète (non requis pour l'installation initiale ou les mises à jour automatiques)
</Windows>

<MacOS>
  * Intel ou Apple Silicon
  * macOS Ventura ou version ultérieure
  * Au moins 4 Go de RAM
  * Privilèges d'administrateur pour l'installation initiale (non requis pour les mises à jour automatiques)
</MacOS>
