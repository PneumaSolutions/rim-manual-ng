Om Remote Incident Manager te kunnen draaien, moet uw systeem voldoen aan de volgende vereisten:

<Windows>
  * Windows 10 (X64) of later
  * Windows Server 2016 (X64) of later
  * Minimaal 4GB RAM
  * Beheerdersprivileges voor volledige functionaliteit (niet vereist voor eerste installatie of automatische updates)
</Windows>

<MacOS>
  * Intel- of Apple-silicium
  * macOS Ventura of later
  * Minimaal 4 GB RAM
  * Beheerdersrechten voor eerste installatie (niet vereist voor automatische updates)
</MacOS>
