Para executar o Remote Incident Manager, seu sistema deve atender ou exceder os seguintes requisitos:

<Windows>
  * Windows 10 (X64) ou posterior
  * Windows Server 2016 (X64) ou posterior
  * Pelo menos 4 GB de RAM
  * Privilégios de administrador para funcionalidade total (não é necessário para instalação inicial ou atualizações automáticas)
</Windows>

<MacOS>
  * Intel ou Apple Silicon
  * macOS Ventura ou posterior
  * Pelo menos 4 GB de RAM
  * Privilégios de administrador para a instalação inicial (não é necessário para atualizações automáticas)
</MacOS>
