import Windows from "~/components/platform/Windows.astro"
import MacOS from "~/components/platform/MacOS.astro"

<Windows>
以下はRIMの完全なキーボードコマンドリファレンスである。グローバルショートカットを持つすべてのコマンドを、それぞれのWindowsフレンドリーなショートカットとともにリストアップする。

| アクション｜コマンド
| ---------------------------------------- | ------------------- |
| RIM メニューにアクセスする｜Windows+Shift+Backspace
| セッションを最小化する｜Windows+Shift+M
| セッションのフリップ｜Windows+Shift+F
| セッションの切断（コントローラまたはターゲット）｜Windows+Shift+D
</Windows>
<MacOS>
以下は、RIMの完全なキーボードコマンドリファレンスです。グローバルショートカットを持つすべてのコマンドを、それぞれのmacOSフレンドリーなショートカットとともにリストアップします。

    アクション｜コマンド
    --- | ---
    RIMメニューにアクセス｜Option+Shift+Backspace
    セッションを最小化｜Option+Shift+M
    セッションの反転｜Option+Shift+F
    セッションの切断（コントローラまたはターゲット）｜Option+Shift+D
</MacOS>
