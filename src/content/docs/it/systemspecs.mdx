Per eseguire Remote Incident Manager, il sistema deve soddisfare o superare i seguenti requisiti:

<Windows>
  * Windows 10 (X64) o successivo
  * Windows Server 2016 (X64) o successivo
  * Almeno 4 GB di RAM
  * Privilegi di amministratore per l'installazione iniziale (non necessari per gli aggiornamenti automatici)
</Windows>

<MacOS>
  * Intel o Apple Silicon
  * macOS Ventura o successivo
  * Almeno 4 GB di RAM
  * Privilegi di amministratore per l'installazione iniziale (non necessari per gli aggiornamenti automatici)
</MacOS>
